# Thar Cuisine – Restaurant Website

**Project Overview**  
Thar Cuisine is a popular local restaurant in Mithi, Pakistan, offering authentic Thari cuisine. This project is a **modern, professional WordPress website** designed to **enhance digital presence, drive inbound calls, and increase online reservations and form submissions**. The website emphasizes a clean, trustworthy, and local-service brand style, including a special **Evening Iftar Discount Offer**.
